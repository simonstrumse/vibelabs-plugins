# Heartbeat Checklist

> What should I check every time I wake up?
> Edit this file to add or remove items. If this file is empty, heartbeat is disabled.

## Tasks
- [ ] Check `TASKS.md` for overdue or critical items
- [ ] Check `ACTIVE_THREADS.md` for pending triggers that are now due

## Calendar
- [ ] Check today's calendar for upcoming events in the next 2 hours
- [ ] Flag any conflicts or prep needed

## General
- [ ] Check `memory/STATE.md` for any time-sensitive watchouts that need action

---

> **How this works:** Every 30 minutes, the assistant reads this file and checks each item.
> If nothing needs attention, it stays silent. If something does, it sends you a message.
>
> **To disable heartbeat:** Delete all checklist items or delete this file.
> **To add custom checks:** Add `- [ ] Check [whatever]` items above.
