# Report skeleton — `crisis-tabletop`

Extends `_core.md`. Universal sections inherited. Mode-specific sections insert between "Proposed revisions" and "Coverage gaps." Sections below include the full AAR (After-Action Review) in Edelman/APCO format.

## Header line

**Run:** <run-id> · **Mode:** crisis-tabletop · **Severity:** <high / critical> · **Rounds:** <2 or 3> · **Personas consulted:** <n> · **Stimulus:** <one-line>
