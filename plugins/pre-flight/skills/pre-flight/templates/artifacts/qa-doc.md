# Artifact: Q&A doc

Anticipated questions with model answers. Industry-standard pre-event artifact in crisis comms and IR prep (NIRI earnings-prep guidance: 20-80 questions). In crisis-pre-flight mode, target is 20-50 questions.

## Structure

```markdown
# Q&A doc — <stimulus title>

**Run:** <run-id> · **Compiled from:** <n> persona reactions · **Approval status:** draft

## Questions by category

<!-- Group by the persona cluster most likely to ask. Within each group, hardest questions first. -->

### Hostile / adversarial (likely from: <activist, adversarial press>)

**Q1.** <Question quoted or paraphrased from a persona's reaction — specifically from the "What would change my position" field of a hostile persona>
- **Model answer:** <answer that addresses the concern without crossing any red_line, without accepting contested framing, and without any phrase in any persona's forbidden_phrases list>
- **Source:** <persona name whose reaction motivated this question>
- **Do not say:** <specific phrases to avoid>
- **Bridge phrase:** <optional — how to pivot to a message-house pillar if the questioner dwells>

**Q2.** ...

### Regulatory / compliance (likely from: <regulator cluster, trade press>)

**Q1.** ...

### Commercial / customer (likely from: <customer cluster, sell-side>)

**Q1.** ...

### Internal / employee (likely from: <internal cluster, if mode includes it>)

**Q1.** ...

### Follow-up questions likely if the first answer is weak

**Q1.** ...

## Questions we deliberately do not have an answer for

<!-- Be honest. If a persona asked something the organization cannot answer (legal, not yet decided, contested), say so here. The spokesperson says "we're not able to comment on that today" — this section documents what those topics are, grounded in the reactions. -->

- **<topic>:** <why this is deliberately unanswered — e.g. "regulator decision pending, commenting would prejudice the process">
- ...
```

## Grounding rules

- Every question is pulled from or motivated by a specific persona reaction. "What would change my position" fields are the richest source.
- Every model answer is checked against every persona's red_lines and forbidden_phrases. A model answer that crosses a red_line is a failure.
- Questions the organization cannot or will not answer are explicitly listed, not omitted.
- Category headers use the actual cluster names from this run's cast, not a generic template.

## Length

20-50 questions. Target: 40 for a 12-persona cast. Each question + model answer block ~80-150 words.
